# 简介

景天：永安当伙计，最爱古董，擅长盗版，梦想成为渝州首富。景天在渝州唐门的最大当铺做伙计，他在乎身份地位，对金钱更是十分看重，但其实内心真正向往的却是自由自在的生活。对剑仙的传说很痴迷，常常幻想有一天能都拜入蜀山门下成为可以御剑飞行的剑仙。
